# Privacidad e independencia

Naubat está diseñado para ofrecer acceso independiente y estructurado a datos de diagnóstico relacionados con batería. La independencia y la transparencia son principios clave.

## Acceso diagnóstico directo

Naubat se comunica directamente con el vehículo mediante protocolos de diagnóstico estandarizados.
El principal es UDS (Unified Diagnostic Services), usado también en talleres profesionales.
No se usan APIs cloud del fabricante ni servicios telemáticos del fabricante.

## Filosofía multimarca

Naubat no está afiliado a ningún fabricante de vehículos.
El sistema está diseñado para operar entre marcas.
La compatibilidad se prioriza por demanda de usuarios y viabilidad técnica.

## Privacidad de datos

Solo se recoge el email del usuario para autenticación y comunicación.
No se almacena VIN.
No se crea un perfil personal de conducción identificable.
No se requiere cuenta de fabricante ni conexión al cloud del fabricante.

## Operación solo lectura

Naubat realiza solicitudes de diagnóstico de solo lectura.
No modifica parámetros.
No realiza configuración, codificación ni flasheo de firmware.

## Por qué importa la independencia

El acceso diagnóstico directo permite análisis transparente de batería.
Un sistema independiente de marca mantiene una metodología consistente entre vehículos.
