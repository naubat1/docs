# Naubat Documentation

Naubat is a mobile app that connects to compatible vehicles through a Bluetooth Low Energy OBD dongle to read diagnostic data and present battery and trip insights.
This documentation is the main reference to understand how the app works in practice.
Its goal is to guide you from installation to daily use, help you interpret what you see in the dashboard, and clarify technical concepts that matter for reliable use.

## What Naubat Is

Naubat is a diagnostic and analytics app focused on electrified vehicles.
It helps you observe vehicle behavior through structured diagnostic signals and presents that information in a usable way.
The app is built to support consistent monitoring of trip and battery-related parameters.

## What This Documentation Covers

- How to install and configure the app
- How to connect and use a compatible dongle
- How app states and data collection behave during real use
- How to read dashboard panels and interpret trip information
- Compatibility limits and practical troubleshooting
- Technical context about vehicles, batteries, and automotive electronics data extraction

## Documentation Objective

This documentation is designed to reduce ambiguity during setup and daily operation.
It explains expected behavior, common edge cases, and practical limitations in clear terms.
It also breaks down key technical topics behind Naubat, including vehicle diagnostics, battery behavior, and how data is obtained from vehicle electronic systems.

## How to Use This Guide

Start with Getting Started for installation and first setup.
Then use Dashboard to understand what each panel shows and how to read trip and aggregated data.
Use Compatibility and FAQ for hardware selection, constraints, and troubleshooting.
